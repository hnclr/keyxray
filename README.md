# KeyXray 🔍🛠️

**KeyXray** is a specialized CLI forensic tool for SSH private keys. Unlike traditional tools that simply validate or reject keys, KeyXray is built with a **forensic-first philosophy**: it actively attempts to rescue corrupted data, audits cryptographic integrity, and hunts for hidden keys in the deepest layers of a system.

Designed for security researchers, forensic analysts, and CTF players, KeyXray bridges the gap between "broken" key fragments and a valid, secure SSH identity.

---

## 🚀 Why KeyXray?

Standard tools like `ssh-keygen` are rigid. If a key has a single hidden character or a slightly malformed header, it is rejected as "invalid." **KeyXray** treats every file as a forensic artifact, analyzing its mathematical properties (Entropy), its structural metadata (Origin Fingerprinting), and providing tools to reconstruct it into a working state.

---

## ✨ Core Capabilities

### 🧪 Advanced Forensics
- **Forensic Reconstruction (Clean):** Radical sanitization of keys corrupted by copy-pasting, email clients, or OCR. Strips internal noise and rebuilds standard PEM blocks.
- **Origin Fingerprinting:** Heuristic analysis to guess if a key was generated by macOS, modern OpenSSH, legacy OpenSSL, or Java-based tools.
- **Heuristic Scoring:** Calculates **Shannon Entropy** to detect weak PRNG (Pseudo-Random Number Generation) and predictable key material.

### 🎯 Deep-Scan Hunting
- **Multi-Layer Scanning:** Recursive directory search with three modes:
  - **Standard:** Fast header check.
  - **Deep:** Full file-content streaming for embedded PEM markers.
  - **Stego:** Entropy-based detection of hidden cryptographic blobs without standard headers.
- **Forensic Carving:** Automatically extract and save discovered keys or high-entropy blobs from within binary files or massive logs.

### 🛡️ Security & Auditing
- **Security Audit:** Automated checks for insecure file permissions (non-0600) and weak cryptographic algorithms (e.g., RSA < 2048 bits).
- **Passphrase Detection:** Instant identification of encrypted private keys.
- **Public Key Extraction:** Regenerate lost public keys directly from the raw material of a private key.

## 🛠 Installation

The recommended way to install **KeyXray** is via `go install`:

```bash
go install github.com/hnclr/keyxray@latest
```

Ensure your `$GOPATH/bin` is in your system's `PATH`.

## 📖 Usage Showcase

### 1. The Forensic "Deep Clean"
Fix a key found in a chat log or a messy document:
```bash
keyxray clean corrupted_key.txt
```

### 2. Hunting Hidden Keys (Steganography)
Find and extract a key hidden inside a binary blob or a log file:
```bash
keyxray hunt /var/log --stego --extract
```

### 3. Comprehensive Audit & Inspection
Get technical specs and security risks in one command:
```bash
keyxray inspect id_rsa
```

### 4. Mathematical Entropy Score
Test if a key was generated with high-quality randomness:
```bash
keyxray score suspicious_device.key
```

### 5. Automated Pairing
Group private and public keys in a messy directory:
```bash
keyxray match ./all_my_keys
```
